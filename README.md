# web-scraping

Install python 3.13.
Cross check the version of the python using the below prompt.
python --version (or) py --version 
Install visual studio code. 
To create a virtual enivironment named venv using this prompt.
You can use windows command prompt or the visual studio code.
python -m venv venv (or) py -m venv venv  
Inside the  activated venv,  
pip install requests 
pip install beautifulsoup4
pip install selenium
pip install pandas
pip install lxml
Run the web scrapping script using,
python scrape.py (or) py scrape.py
After running the script you will get, the csv file,excel file and database file.
